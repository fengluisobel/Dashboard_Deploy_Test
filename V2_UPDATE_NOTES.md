## 🆕 新功能概览

### 1️⃣ 指标翻转卡片 (Flip Cards)

**功能描述**: 核心KPI卡片支持鼠标悬停翻转,正面显示指标值,背面显示详细说明

**正面显示**:
- 指标名称(中英文)
- 指标当前值
- 关注该指标的角色标签
- 提示文字

**背面显示**:
- 📖 定义: 该指标的业务含义
- 🧮 公式: 计算方法
- 📊 构成: 数据来源和组成部分
- 🎯 基准: 优秀/良好/需改进的参考值
- 👥 关注角色: 哪些角色应重点关注此指标

**使用方法**:
- 鼠标悬停在KPI卡片上即可自动翻转
- 移开鼠标自动翻回正面

**降低学习成本**:
- 新用户可快速了解每个指标的含义
- 无需查阅外部文档
- 一秒即可获得完整上下文

### 2️⃣ 四大角色视角 (Role-Based Views)

**功能描述**: 根据用户角色提供个性化的Dashboard视图和关键指标

**关注重点**: 战略指标、成本控制、整体效能

**核心KPI**:
- 平均招聘周期
- 单次招聘成本
- 招聘顾问人效
- 试用期转正率
- 候选人NPS

**典型使用场景**:
- 月度/季度战略复盘
- 预算分配决策
- 团队效能评估
- 向CEO汇报招聘成果

---

#### 📊 角色2: HRD (人力资源总监)

**典型使用场景**:
- 流程瓶颈识别

- 业务满意度提升

### 3️⃣ AI洞察助手 (AI Insights)

**功能描述**: 自动分析关键指标数据,提供智能洞察和行动建议

#### 🎯 AI洞察功能

**洞察卡片 (橙色)**:
- 💡 数据洞察标题
- 指标当前状态(🟢正常 / 🟡警告 / 🔴异常)
- 与基准值对比
- 问题根因分析
- 风险提示

**行动卡片 (蓝绿色)**:
- 🎯 推荐动作标题
- ⚡ 立即行动: 紧急优先级行动
- 💡 建议: 中期优化建议
- ✅ 持续/保持: 长期巩固措施
- 📋 后续跟进: 监控要点

#### 📊 覆盖的关键指标

1. **平均招聘周期**
   - 洞察: 识别周期过长问题
   - 建议: 分析瓶颈环节,建立快速通道

2. **试用期转正率**
   - 洞察: 发现质量问题信号
   - 建议: 优化面试标准,强化培训

3. **录用接受率**
   - 洞察: 分析Offer被拒原因
   - 建议: 薪酬对标,期望管理

4. **单次招聘成本**
   - 洞察: 成本结构分析
   - 建议: 渠道优化,降本增效

5. **候选人NPS**
   - 洞察: 候选人体验问题
   - 建议: 流程优化,面试官培训

#### 🤖 AI分析逻辑

```python
if 指标值 > 红线阈值:
    🔴 严重问题 → ⚡ 立即行动
elif 指标值 > 黄线阈值:
    🟡 需要改进 → 💡 优化建议
else:
    🟢 表现良好 → ✅ 保持措施
```

**智能化特点**:
- 自动识别异常
- 分级预警(红/黄/绿)
- 提供具体行动方案
- 关联数据支撑

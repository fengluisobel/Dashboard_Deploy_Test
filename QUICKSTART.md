# 快速开始指南

## 🚀 5分钟快速启动

### 第一步: 确认环境

确保你已安装Python 3.8或更高版本:
```bash
python --version
```

### 第二步: 安装依赖

在项目目录下运行:
```bash
pip install streamlit pandas numpy plotly
```

或使用requirements文件:
```bash
pip install -r requirements.txt
```

### 第三步: 启动应用

```bash
cd "E:\AI Staff\AI_Hire_Dashboard"
streamlit run recruitment_dashboard.py
```

### 第四步: 访问应用

浏览器会自动打开,或手动访问: `http://localhost:8501`

## 🔧 故障排除

### 问题1: 应用无法启动
**解决方案**:
- 检查Python版本是否>=3.8
- 确认已安装所有依赖: `pip list`
- 查看错误日志,定位具体问题

### 问题2: 图表不显示
**解决方案**:
- 清除浏览器缓存
- 检查数据是否为空
- 调整筛选条件

### 问题3: 数据导出失败
**解决方案**:
- 检查文件写入权限
- 确认磁盘空间充足
- 尝试更换导出路径

### 问题4: 性能缓慢
**解决方案**:
- 减少筛选的数据量
- 关闭其他占用资源的程序
- 考虑使用数据采样

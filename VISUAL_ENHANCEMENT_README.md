## 🔄 性能优化

### CSS优化
- 使用 `transform` 而非 `top/left` 做动画
- 启用 GPU 加速（`transform3d`, `backdrop-filter`）
- 减少重绘和回流

### 动画优化
- 使用 `cubic-bezier` 缓动函数
- 合理的动画时长（0.2s-0.6s）
- 避免同时触发大量动画

### 加载优化
- Google Fonts 异步加载
- CSS 内联注入
- 最小化重复样式

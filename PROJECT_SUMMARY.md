## 💡 最佳实践

### 使用建议

1. **定期复盘**
   - 建议每月导出数据进行月度复盘
   - 关注红色预警指标,及时采取行动
   - 对比环比/同比,发现趋势变化

2. **指标联动分析**
   - 成本高 + 转化低 → 渠道优化
   - 招聘快 + 离职高 → 质量问题
   - 满意度低 + NPS低 → 体验改进

3. **数据驱动决策**
   - 用数据验证假设
   - 用趋势预测未来
   - 用对比发现差距

### 常见分析路径

**路径1: 招聘效率诊断**
```
速度Tab → 查看TTF趋势 → 分析阶段耗时 → 识别瓶颈 → 制定改进措施
```

**路径2: 招聘质量评估**
```
质量Tab → 查看转正率 → 分析离职率 → 对比绩效分布 → 优化招聘标准
```

**路径3: 渠道ROI优化**
```
转化Tab → 查看渠道矩阵 → 成本Tab → 对比渠道单价 → 调整渠道组合
```

**路径4: 候选人体验改进**
```
体验Tab → 查看NPS趋势 → 分析雷达图 → 识别短板 → 制定改进计划
```
